# 🌿 AI-Based Crop Disease Detection

## 📌 Overview
This project detects crop diseases using deep learning models like CNN, MobileNetV2, and ResNet50.

## 🚀 Features
- Image-based disease detection
- Multiple model comparison
- Real-time prediction system
- Accuracy visualization

## 📂 Dataset
PlantVillage dataset (Tomato & Potato)

## 🛠 Technologies
- Python
- TensorFlow / Keras
- Google Colab

## 📊 Models Used
- CNN
- MobileNetV2
- ResNet50

## 📸 Output
Input: Leaf Image  
Output: Disease Prediction

## 🔮 Future Scope
- Mobile App integration
- Real-time camera detection
